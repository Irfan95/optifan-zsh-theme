# Optifan ZSH Theme
This repository contains the custom theme I have created for my liking to use in ZSH.

## How to install
To install this theme I recommend placing the .zsh-theme file in the themes folder under your .oh-my-zsh directory as per below
~/.oh-my-zsh/themes/optifan.zsh-theme

Once placed in this directory you can update your .zshrc file by updating the default theme variable with the following:
```shell
ZSH_THEME="optifan"
```

Once you have updated this you can update your shell on the spot by running the following:
```shell
exec zsh
```
